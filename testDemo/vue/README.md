# 前端相关依赖

### npm 下载
```
npm install
```

### cli 下载
```
npm install -g @vue/cli
```

### 组件库下载
```
npm install element-plus --save
```

### 编译启动
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
