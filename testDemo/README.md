##·注意事项
·配置好相关依赖

·创建数据库 sustc ,导入ddl.sql文件(登录功能请手动创建 user表)

·将 application.properties 中的 数据库用户名，密码，fileIO包下文件路径替换为本地信息

·确保 端口8081(前端)、9090(后端)保持空闲状态

·若需启用局域网内访问，请修改vue.config.js 中的ip,并开放防火墙
